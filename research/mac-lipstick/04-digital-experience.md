# MAC 디지털 경험 & 웹사이트

## 디지털 전략 개요

MAC은 이커머스 사이트를 단순한 쇼핑몰이 아닌 **브랜드 매거진**으로 재정의하고 있다. 검색 중심 쇼핑에서 영감·룩 기반 발견(discovery) 경험으로 전환하는 것이 핵심 전략.

---

## 웹사이트 리디자인 (2024~)

### MediaMonks × MAC Digital Flagship

| 항목 | 내용 |
|------|------|
| **에이전시** | MediaMonks |
| **크리에이티브 디렉터** | Alex Otto |
| **디자이너** | Gaia Zeyp, Yanchen Zhao |
| **UX 디자이너** | Anna Chau |

### 디자인 컨셉

> "M·A·C's new strategy transforms the e-commerce platform into a branded magazine"

**3대 브랜드 필러:** Punk · Vibrant · Authoritarian

### 핵심 디자인 요소

1. **밝은 컬러 + 고대비 타이포** — 화장품 세계에서의 자신감 표현
2. **대형 로고 배치** — M·A·C 로고를 지배적 스케일로
3. **키네틱 타이포그래피** — 동적 텍스트 템플릿 세트
4. **모듈러 시스템** — 매거진형 UI 컴포넌트
5. **3D 모션** — Cinema 4D, After Effects 활용

### 타이포그래피 변화

| Before | After |
|--------|-------|
| Trade Gothic | **Ano Bold** |

→ 웹뿐 아니라 OOH, 로고 적용, 컬러 선택 전반에 영향

## 주요 디지털 기능

### 1. Looks Destination

| 항목 | 내용 |
|------|------|
| **개념** | 일상·아이코닉·트렌딩 룩 기반 쇼핑 |
| **구성** | 스텝바이스텝 가이드 + 쇼핑 가능 |
| **효과** | 평균 주문 금액(AOV) 유의미한 증가 |
| **연계** | Club MAC 컨셉 스토어 디지털 스토리텔링 |

### 2. Color Play & Brush Finder

- 대규모 제품 기반 인터랙티브 앱
- 컬러·브러시 탐색 도구

### 3. Virtual Try-On (버추얼 트라이온)

- 200+ 립 컬러
- 3단계로 빠른 체험
- 집에서 AR 체험 가능

### 4. How-To Media Player

- 프로 아티스트 따라하기 영상
- 큐포인트 제품 프레젠테이션
- 아티스트 주도 콘텐츠

### 5. Gallery Module

- 브랜드 이미지를 리치 인터랙티브로 랜딩페이지에 전시

## 한국 공식 사이트 (maccosmetics.co.kr)

### 립 카테고리 페이지 구조

```
립 (Lips)
├── 립스틱
├── 리퀴드 립
├── 립 글라스
├── 립 라이너
├── 미니 맥
├── 립 팔레트 + 키트
└── 립 케어 + 프라이머
```

### 주요 UX 패턴

- **제품 카드:** 컬러 스와치 + 가격 + 피니시 설명
- **"색상 더보기"** — 다중 컬러 선택 UI
- **"메이크업 체험하기"** — AR/버추얼 트라이온 CTA
- **셀럽 원픽:** ELLA, DOJA CAT 등 아티스트 추천
- **립 콤보:** 립스틱 + 립 펜슬 매칭
- **컬러 스펙트럼:** 히어로 컬러별 스토리텔링 섹션

### 카피라이팅 톤 (한국)

- 짧고 리듬감 있는 3단어 수식: "아이코닉. 대담한. 세계적인 인기."
- 기능 + 감성 병행: "12시간 동안 생생하게 표현되는 컬러"
- 행동 유도: "지금 체험하기", "쇼핑하기", "립 콤보 쇼핑하기"

## 소셜 & 디지털 마케팅

### TikTok 전략

- **립 콤보** 트렌드 적극 활용
- 90년대 립 펜슬 컬러 재조명
- 크리에이터 자유로운 제품 활용 장려
- TikTok Shop 진출
- CreatorIQ 기준 상위 10개 메이크업 브랜드 (earned media value)

### 소셜 미디어 포지션

- MAC 아티스트 13,000명이 소셜 콘텐츠 생산
- "쿨 키즈에게 맡기고, 망가뜨리고, 자기만의 것으로 만들게 한다"
- 어스피레이셔널 캠페인 + UGC의 균형

### 글로벌 디지털 크리에이티브 시스템

- 28+ 히어로 제품용 모듈러 시스템
- 글로벌 캠페인 → 브랜드 사이트·이메일·소셜·리테일 파트너 일관 적용
