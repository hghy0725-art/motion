# MAC 캠페인 & 문화적 영향력

## Viva Glam — MAC의 상징적 캠페인

### 개요

| 항목 | 내용 |
|------|------|
| **시작** | 1994년 |
| **목적** | HIV/AIDS 프로그램 기금 모금 & 인식 제고 |
| **기부** | 판매 수익 100% 기부 |
| **누적 기부** | $545 million+ (2026 기준) |
| **확장** | 성별·인종·환경 평등 지원으로 확대 |

### 역사적 의미

- **업계 최초:** 메이저 뷰티 브랜드가 퀴어·성소수자 앰버서더(RuPaul)를 전면에 내세운 캠페인
- **1994년 배경:** HIV/AIDS 스티그마가 극심했던 시기, LGBTQIA+ 커뮤니티 타격
- **브랜드 정체성의 핵심:** Viva Glam = MAC이 무엇을 믿는지의 증거

### 슬로건 해석

> **"Viva"** = 삶을 살다 (to live life)  
> **"Glam"** = 화려하게 (to be glamorous)  
> = "삶을 화려하게, 긍정적으로 살다"

### 캠페인 개요

| 항목 | 내용 |
|------|------|
| **앰버서더** | Chappell Roan (가수) |

| **스타일** | Genesis Webb |

### 제품 라인업

| 제품 | 컬러 | 설명 |
|------|------|------|
| **Unnatural Red Head** | 다크 체리 매트 | Chappell의 헤어 컬러 매칭 |
| **Roan of Arc** | 미드톤 플럼 립밤 | — |
| **Damnsel** | 시어 레드-플럼 글로스 | 골드 쉬머 |

### 비주얼 컨셉

**두 페르소나:**
1. **"The Hero"** — 하이 드래그, 극적
2. **"The Empress"** — 아티스틱, 라인 워크 중심

**영감 소스:**
- 1920년대 프랑스 극장
- 초현실주의 오브제
- 아르누보·메디벌 문장(crest) 디자인
- 1940년대 영화의 과장된 황제 이미지

**메이크업 룩:**
- 플럼·골드·루비·터콰이즈 드라마틱 아이
- 페더리 래시, 가는 눈썹
- 웨이브진 레드 컬
- Unnatural Red Head + Vino 립 펜슬 콤보

### Chappell Roan의 말

> "이 캠페인은 포용적이고, 다른 종류의 글래머를 대표하도록 의도되었습니다. 이상한 아트 키즈와 다르고, 드라마틱하고, 대담한 사람들이 자신을 여기서 발견했으면 합니다."

## 40주년 캠페인 (2024)

### 주요 메시지

- 40년 헤리티지 + 차세대 진화
- "All Ages, All Races, All Genders" 재확인
- Macximal 라인 런칭 = 제품 혁신의 상징
- Viva Glam 30주년 동시 기념

### 전략 키워드

1. **Nowstalgia** — 과거의 MAC을 현재 트렌드로 재해석
2. **Club MAC** — 커뮤니티 중심 체험형 리테일
3. **Artistry First** — 아티스트 네트워크 재강조
4. **Digital Native** — TikTok·소셜 커머스 확장

## 캠페인 비주얼 레퍼런스 (디자인 참고)

### 반복되는 비주얼 패턴

1. **대담한 원색** — 레드, 옐로, 블루 등 프라이머리
2. **드래그·퀴어 에스테틱** — Viva Glam의 시그니처
3. **아티스트 라인 워크** — Andrew Dahling 스타일
4. **역사적 오마주** — 1920s 극장, 1940s 영화, 아르누보
5. **립스틱 = 히어로** — 제품이 화면의 절대 주인공
6. **타이포 = 립스틱** — 립스틱 형태에서 영감받은 타이포그래피
7. **포용적 글래머** — "다른 종류의 글래머"
